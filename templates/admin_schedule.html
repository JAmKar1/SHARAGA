Инструкция по загрузке PDF
  1. Перейдите на страницу Расписание
  2. Вверху страницы будет форма загрузки PDF файла
  3. Загрузите PDF файл с расписанием
  4. Система автоматически распарсит файл и сохранит расписание в БД
  5. После загрузки расписание сразу будет доступно всем пользователям
Важно: PDF файл должен содержать названия групп в формате "ИСП-101", "ПИ-21" и т.д.
